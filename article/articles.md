# 文集基本信息

## 获取文集基本信息

> https://api.bilibili.com/x/article/list/web/articles

*请求方式：GET*

认证方式：我不确定，dalao你帮我写吧

**url参数：**

| 参数名 | 类型 | 内容     | 必要性 | 备注 |
| ------ | ---- | -------- | ------ | ---- |
| id     | num  | 文集rlID | 必要   |      |

**json回复：**

根对象：

| 字段    | 类型 | 内容     | 备注                                            |
| ------- | ---- | -------- | ----------------------------------------------- |
| code    | num  | 返回值   | 0：成功<br />-400：请求错误<br />-404：无此信息 |
| message | str  | 错误信息 | 默认为0                                         |
| ttl     | num  | 1        |                                                 |
| data    | obj  | 信息本体 |                                                 |

`data`对象：

| 字段              | 类型   | 内容         | 备注   |
| ----------------- | ------ | ---------- | -------|
| list              | obj    | 文集概览    |        |
